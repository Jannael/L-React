# React Learning Projects

This repository contains a collection of projects and exercises created for learning React and related technologies (Framer Motion, GSAP, Tailwind CSS, etc.). Each folder represents a specific lesson, component, or full application.

Below is an index of all the projects included in this repository to help you easily locate code examples.

## Project Index

| Folder | Project Name | Description |
| :--- | :--- | :--- |
| **01** | Calculator Display | A simple display component demonstrating basic state management and props. |
| **02** | Sidebar Navigation | An expandable sidebar menu with navigation items and icons. |
| **03** | Animated Modal | A modal component featuring entrance and exit animations using Framer Motion. |
| **04** | Theme Switcher | A toggle button to switch between light and dark themes using CSS variables. |
| **05** | Basic Router Setup | A minimal example of setting up client-side routing with React Router. |
| **06** | Product Card Effect | An interactive product card (Nike Shoe) with image switching on hover and backdrop effects. |
| **07** | Character Gallery | A full-screen character selector (Dragon Ball) that changes the background image on hover. |
| **08** | Travel Slider | A hero section with a slider for exploring different countries (World Travel). |
| **09** | Interactive Bear | An SVG animation of a bear that tracks the mouse cursor movement (eyes and head). |
| **10** | Typing Game | A keyboard training game that highlights keys and tracks user input. |
| **11** | Parallax Effect | A simple implementation of a parallax scrolling effect on an image. |
| **12** | GSAP Scroll Animation | An animation sequence triggered by scrolling, featuring Dragon Ball assets. |
| **13** | Login/Signup Form | A stylish split-screen login and signup form with a sliding transition panel. |
| **14** | Weather App | A functional weather application fetching data from OpenWeatherMap API. |
| **15** | Pokedex | A Pokemon search tool connecting to the PokeAPI to display Pokemon details. |
| **16** | Image Carousel | A card-deck style image carousel using the Swiper library. |
| **17** | TMNT Landing Page | A themed landing page featuring Teenage Mutant Ninja Turtles components (Villains, Mutagen). |
| **18** | Liquid Loader | A creative loading animation simulating liquid filling up a flask. |
| **chatbot-project** | Chatbot | A conversational interface with message history and input handling. |
| **chess** | Chess Game | A fully playable Chess game logic with move validation and board rendering. |
| **ecommerce** | E-commerce Store | A multi-page e-commerce application with a home page, checkout, tracking, and order history. |
| **basicexercises** | HTML Experiments | A folder containing basic HTML/CSS experiments and exercises. |
